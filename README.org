* fmt-examples
Просто небольшая "шпаргалка" по использованию fmt в C++ с разными типами данных.

** main.cpp
Демонстрирует возможности форматирования с помощью ~fmt~:

- =int= — целые числа
  - Автоматический вывод
  - Шестнадцатеричный формат (={:x}=)
  - С префиксом =0x= (={:#x}=)
  - С нулевым заполнением до 6 символов (={:#06x}=)

- =std::string= — строки
  - Автоматический вывод
  - Выравнивание по правому краю с шириной (={:>10}=)

- =double= — числа с плавающей точкой
  - Автоматический вывод
  - Ограничение точности (={:.2f}=)
  - Научная запись (={:e}=)

- =bool= — логический тип
  - Вывод как =true=/=false=
  - Тернарный оператор для текстового представления

- =char= — символы
  - Вывод символа
  - Преобразование в код символа (=static_cast<int>=)

- Множественные аргументы
  - Подстановка нескольких значений в одну строку
  - Индексированные аргументы (={2}= ={0}= ={1}=)
